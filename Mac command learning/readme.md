# remove/delete a folder (directory) using the terminal: rm -r folder_name

```
rm -r folder_name
rm -rf folder_name
mv folder_name ~/.Trash/
trash folder_name
```


* `rm` → remove
* `-r` → recursive (removes all files & subfolders inside)
* `-f` → force (skips confirmation prompts)
